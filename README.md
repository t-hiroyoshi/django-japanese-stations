# django-jp-stations

django-jp-stations is a simple japanese stations apps.

## Quick start

* Add "jp-stations" to your INSTALLED_APPS setting like this

```python
INSTALLED_APPS = [
  ...
  'jp_stations',
]
```

* Run `python manage.py migrate` to create the stations models.
